AdvSpec
=======

Advanced Spectator plugin for TF2

Usage
-
Build the plugin by following `BUILD.md`, then copy advspec.dll to your TF2 server's addons folder.  
Load it via `plugin_load addons\advspec`, or create a VDF file.
